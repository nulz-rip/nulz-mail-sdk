# Contributing

Thanks for considering contributing to the Nulz Mail SDK.

## How to contribute

1. **Fork** the repo on GitHub.
2. **Clone** your fork and create a branch:
   ```bash
   git clone https://github.com/nulz-rip/nulz-mail-sdk.git
   cd nulz-mail-sdk
   git checkout -b your-feature-or-fix
   ```
3. **Install** and run the unit tests:
   ```bash
   npm install
   npm run test:unit
   ```
4. **Make your changes.** Keep the code style consistent and add tests where it makes sense.
5. **Commit** with a clear message, then **push** to your fork.
6. **Open a Pull Request** against the main repo. Describe what you changed and why. A maintainer will review.

For release and publish steps, see [MAINTAINERS.md](./MAINTAINERS.md) (maintainers only).
